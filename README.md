# Car Dealership Inventory System

## Project Overview

A full-stack inventory management system built using FastAPI, React, Tailwind CSS, SQLite, JWT Authentication, and Test-Driven Development (TDD).

## Technologies Used
- FastAPI
- SQLite
- SQLAlchemy
- JWT
- React
- Tailwind CSS
- Axios
- Pytest

## Features
- User Registration
- Login
- Vehicle CRUD
- Search
- Purchase
- Restock
- JWT Authentication
